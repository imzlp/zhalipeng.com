UE4 Source Code

下面列出的是EpicGames/UnrealEngine仓库的分流地址，国内的网络环境直接从Github上直接clone实在太慢了，在VPS上几分钟搞定。

注意：Epic不允许UE代码的二次分发，所以下载下列代码之前请确保你同意UE的EULA协议。

另外，在下载后，最好使用git fetch更新一下，保持分支为最新（更新就很快了）。

UE4 Toolchain

  • UE4 Launcher 方便启动UE的工具
  • ue4program 创建UE的StandaloneApplication工具
  • UE4_API_FOR_DASH 抓取UE4 API生成的带索引Dash文档
  • UnrealPakViewer 查看UE4 Pak文件的工具
  • UE4-Binary-Builder 从源码编译UE4的工具
  • UModel 预览和导出未加密Pak的资源
