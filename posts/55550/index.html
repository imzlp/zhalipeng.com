使用Github来管理博客源文件

发表于 2014-11-20 23:52 分类于 博客管理 ， Hexo
本文字数： 866 阅读时长 ≈ 2 分钟

将博客的源文件(非hexo生成的文件)托管到Github上，方便同步和控制版本。

  1. 在github上建立一个新的repository，名称blog
  2. 在本地进入blog文件夹，使用命令git init创建仓库
  3. 设置远程仓库地址，并更新
  1. 修改.gitignore文件（如果没有，请创建），在里面加入public/和.deploy/，因为这两个文件夹是每次generate和deploy都会更新，对我们没用，因此忽略这两个文件的更新。
  2. 使用命令git add .，将所有文件提交到缓存区。
  3. 使用命令git commit -m “add all files” ，将这些文件提交到本地仓库。
  4. 使用命令git push origin master，将本地仓库的改动推送到github仓库。
